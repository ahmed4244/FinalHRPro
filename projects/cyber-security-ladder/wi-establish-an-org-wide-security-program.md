# WI — Establish an org-wide security program

Stand up a security program that raises the whole org — a **secure SDLC**, or a
**zero-trust** rollout — with standards teams actually follow and a way to measure
the risk it removes.

- **Done when:** the program is live across the org and its impact is measured.
