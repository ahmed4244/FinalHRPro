# A checklist beats memory at the end of a long day

Tired eyes skim a typo; a check does not. The same list, run the same way,
gives the same answer at 09:00 and at 23:00 — that is what deterministic
means, and it is why the doctor reads every note instead of trusting my
evening attention.
