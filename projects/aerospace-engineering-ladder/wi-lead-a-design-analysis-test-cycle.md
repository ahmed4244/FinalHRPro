# WI — Lead a design–analysis–test cycle

Take a component through a full **design → analysis → test → correlate** loop:
model it, predict its behavior, test it, and reconcile the results to sign-off.

- **Done when:** test data matches (or corrects) the analysis and the component is signed off.
