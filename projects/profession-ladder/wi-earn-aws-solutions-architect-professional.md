# WI — Earn AWS Solutions Architect – Professional (SAP-C02)

The professional-tier cloud cert for the Staff rung — it explicitly covers design
for **organisational complexity**, migration planning and cost control, which is
exactly the multi-team surface Staff work lives on. Builds on the associate cert
from the Senior rung ([[wi-earn-aws-solutions-architect-associate]]).

- **Done when:** SAP-C02 passed (or the GCP Professional Cloud Architect equivalent
  for a GCP shop).
- **Optional track:** iSAQB CPSA-Advanced for a pure software-architecture path.
