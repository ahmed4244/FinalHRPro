# The Cyber Security career ladder

The ten-year map for the **Cyber Security** cluster. You start with a bachelor's
degree as a **Security Engineer**; the ladder climbs one rung every five years:

| Year | Rung | The jump |
|---|---|---|
| 0 (now) | Security Engineer | you are here — a bachelor's, ~0 yrs |
| +5 | Senior Security Engineer | own a security domain end-to-end |
| +10 | Staff / Principal Security Engineer | set security direction across the org |

Rung 1 is [[senior-security-engineer]]; rung 2 is [[staff-security-engineer]].

## The axis every rung moves along
Like every engineering ladder, the jump is **scope**
([[the-scope-ladder-task-to-org]]): a Security Engineer works alerts and controls;
a Senior owns a whole security domain; a Staff/Principal engineer sets the security
architecture and standards the whole org builds on. And as with any promotion, you
earn it by [[promotion-is-operating-at-the-next-level|already operating at the next
level]] — the incidents led, the risks driven down, the standards adopted.

- [[staff-security-engineer]] — the second promotion (+10 yrs)
