# WI — Write an RFC the team adopts

Author a design doc / RFC that the team actually builds against, and drive the
consensus to get there. This is the written half of Senior: showing you can shape a
decision, not just implement one.

- **Done when:** the RFC is approved and a real project is built from it.
- **Backed by:** the study in [[wi-study-system-design-and-ddia]]; often the doc
  behind [[wi-lead-a-team-scoped-project]].
