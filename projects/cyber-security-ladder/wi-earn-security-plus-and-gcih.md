# WI — Earn Security+ then GIAC GCIH

The cornerstone certs for the Senior rung: **CompTIA Security+** as the baseline,
then a hands-on **GIAC GCIH** (incident handling) — or **OSCP** for the offensive
track. The credential that says "I can run security work, not just watch alerts."

- **Done when:** Security+ passed and GCIH (or OSCP) earned.
