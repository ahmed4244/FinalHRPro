# WI — Grow senior security engineers

At Staff/Principal you **sponsor**, not just mentor: level up senior security
engineers, argue their cases, and drive the org's security standards through talks
and design-review forums.

- **Done when:** you've sponsored an engineer toward promotion and own a standard the org follows.
